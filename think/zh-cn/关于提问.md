人会提出哪些问题:
是什么
为什么 -> 原因是什么
怎么办 -> 方法是什么
哪里 -> 地点是什么、 问题点是什么
什么时候 -> 条件是什么
谁 -> 对象是什么
是或否问题: 对不对、好不好

围绕问题的三个能力: 提出问题, 解答问题,运用答案

一旦我们表达出来一句话 “xx是XX”, 那么这句话通常都能判断对错,当然存在例外,比如悖论

一旦一句话可以判断对错,那就存在一个逆否命题和他对错保持一致.

当我们知道了一个东西的原因.我们就可以尝试把原因当成目的

当我知道了一个东西的办法是什么,我们就可以尝试设计出目的组

每个人都在一定程度上会用逆否命题,比如把手伸到火旁边,马上感觉到烫,下一次就再也不会这样做了

每个人都在一定程度上会用三段论, 比如老师跟学生说: 所有人都安静. 大家都知道自己要安静.

即使我们知道了逆否命题和原命题的关系，实际生活中也是不够用的，因为我们还得知道一个命题怎么转化成逆否命题,当然怎么转化也是有语法上的规律的，但是这些语法的上的规律似乎不能直接用,只能在事后用来检查我们转化的对不对.
有些时候一些规律是不能直接用的,比如主谓宾,当我说完一个谓语后.我不可能在所有的宾语中找答案.
很多规律的用法其实是在事后检查输入对不对用的, 是一套自我奖惩机制,类似强化学习.
虽然不能在所有宾语中选一个答案,但是依然可能有多个候选项,所以其实这些规律可能也存在直接用的情况

问题的种类:
- A的B是什么
- A是什么
- A是不是B

三段论:
- A -(p)->  B
- C -(q)-> A
- 所以: C -(p)-> B

当我们在使用"的"这个连接词的时候, 比如: A的B, 可以把他理解成在B向量的基础上加上A向量,也就是: B + A.
