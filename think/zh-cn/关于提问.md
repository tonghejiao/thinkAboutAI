人会提出哪些问题:
是什么
为什么 -> 原因是什么
怎么办 -> 方法是什么
哪里 -> 地点是什么、 问题点是什么
什么时候 -> 条件是什么
谁 -> 对象是什么
是或否问题: 对不对、好不好

围绕问题的三个能力: 提出问题, 解答问题,运用答案

一旦我们表达出来一句话 “xx是XX”, 那么这句话通常都能判断对错,当然存在例外,比如悖论

一旦一句话可以判断对错,那就存在一个逆否命题和他对错保持一致.

当我们知道了一个东西的原因.我们就可以尝试把原因当成目的

当我知道了一个东西的办法是什么,我们就可以尝试设计出目的组

每个人都在一定程度上会用逆否命题,比如把手伸到火旁边,马上感觉到烫,下一次就再也不会这样做了

每个人都在一定程度上会用三段论, 比如老师跟学生说: 所有人都安静. 大家都知道自己要安静.

即使我们知道了逆否命题和原命题的关系，实际生活中也是不够用的，因为我们还得知道一个命题怎么转化成逆否命题,当然怎么转化也是有语法上的规律的，但是这些语法的上的规律似乎不能直接用,只能在事后用来检查我们转化的对不对.
有些时候一些规律是不能直接用的,比如主谓宾,当我说完一个谓语后.我不可能在所有的宾语中找答案.
很多规律的用法其实是在事后检查输入对不对用的, 是一套自我奖惩机制,类似强化学习.
虽然不能在所有宾语中选一个答案,但是依然可能有多个候选项,所以其实这些规律可能也存在直接用的情况

问题的种类:
- A的B是什么 -> A向量在B向量的投影是什么
- A是什么 -> A向量在很多向量上的投影是什么
- A是不是B -> A向量在B上有没有投影

三段论:
- A -(p)->  B
- C -(q)-> A
- 所以: C -(p)-> B

当我们在使用"的"这个连接词的时候, 比如: A的B, 可以把他理解成在B向量的基础上加上A向量,也就是: B + A.
在中文里"的"通常有两种意思,一个代表所属关系(A的B,代表B属于A),一个代表形容的关系(A的B, A是B的形容词, 如 漂亮的树)

A是B ,会造成两个不一样的向量联系在了一起,而且A是B 不代表B是A.

当A和B的关联是C 的时候,我一般可以理解成A向量在B向量的投影是C

我们看到一只猴子能上树,我们大脑可能会给猴子一个编码A,树一个编码B 上一个编码C.同时大脑中还储存了A -> 猴子的各种特征 B -> 树的各种特征 这种映射关系. 但是如果我们选择性忽略这种映射关系, 那我们天然就得到了一句抽象的话 比如: A 能上树.或者 A能上 B 等等. 这可能是我们能举一反三和具有抽象思维的原理.

我们人类应该天生有想要搞懂一切的欲望,比如搞懂文字,看懂别人说的话.这些欲望在我们的大脑中可能天生就有对应的编码,但是我们是如何学会正确表达出自己的欲望的?我们提出的疑问本身是对求知欲的一种描述.我们会在预测出现误差的时候产生求知欲,提问可能只是一种满足求知欲的方式,只是我们发现这种方法非常好用,所以基本上一直在使用.即使欲望一直得不到满足,应该也需要一个机制去降低欲望,否则他就无法执行更重要的事了.

另外求知欲是高度抽象的,大脑里应该有一个把求知欲转化成一个更加具体的目的过程,这个过程可能需要根据当前的场景+求知欲一起生产出来比较具体的目的.

其实每种欲望可能都是一个数值(代表欲望的强烈程度),他们同时存在,只不过有时候有些数值比较大,就显得好像当时我们只有这一个欲望.而且数值本身不是关键,关键是怎么降低数值,什么时候能降低数值.我们大脑可能先天就有一套降低数值的机制.这套机制是什么呢?
通常是和预期不一致的时候,这种预期包含预测和期待,比如我有一个目的没有得到满足.或者现实和预测不一致,总之都有一个比较后发现不一致的情况.

当我们使用语言描述内心的时候,说出来的话既要符合语法的规则,又要准确地描述内心,这可能意味着,有两组神经网络都在审视这句话,一个用来判断这句话是否符合语法,另一个判断是否准确描述内心.

问为什么可能是一件比较简单的事情，通常的格式是 为什么xx 这里的xx通常是我们感受到的场景，比如看到的东西。为什么会这样。

问怎么做可能也类似，通常也是 怎么会xx 或者怎么才能达到xx目的。

当前额叶提出一个问题，比如 他到底是什么目的？ 前额叶把这个问题丢给基底节，基底节再把它散给下级区域，最终他们又会把信息压缩汇总返回到前额叶，前额叶再得出一个结论，他的目的是xxx，这个过程本身就是把几乎所有脑区的权重都训练了一遍。当然这可能不是一遍就能训练成的。
事实上我们要分析一个人的目的是比较困难的, 常常需要结合所见所闻,再加上换位思考,再加上以前的知识,综合判断出一个结论: 他到底是怎么想的.这种综合判断能力看起来就非常像是上述的那样,大脑中任何一个单一的模块恐怕都无法作出准确的判断.
那么同样的,流程的记忆又是存在哪里的呢,我们不妨分析一下,如何判断下一步该怎么做,事实上这也是一件非常难的事情.也是常常需要根据当下的情况+以往的知识综合判断. 当然似乎有些情况会无脑一点,当下的情况占比可能会比较低

语言中常常有一些泛指的词,比如它,比如第一个数,第二个数.他们根据当时的情况指向不同的东西.能理解这些词,是我们执行某些流程的关键,比如两个数相乘.这两个数随便换,我们也能计算出它的结果.
事实上我们可以把任何几个东西看成一个整体,符合某些特征把它看成一个人,符合某些特征就看成一条狗,看成一个房子等等,似乎本质上是一种多对一的能力,或者说它可能是一种通用的压缩算法

假如设计一个游戏: 在一个篮子里有很多各种形状的纸片, 要求是把一个三角形,一个正方形,一个圆形分成一组,数一数一共有多少组?
只要我们听懂了这个游戏规则,我们就能进行正确的分组,不管怎么变化. 

如何快速的进行随意的分组?如何快速的进行注意力随意的集中?这两个快速的动作是怎么实现的?
大脑其实并不是所有脑区的神经元权重都无法快速调整,有些时候人是需要快速记忆的,这样才能适应这个可能快速变化的环境,当然也不是所有脑区都允许这样,可能因为这样很耗能.
